Integración de Docker para Tuxeria

14 de diciembre de 2025

Para facilitar las demos y mantener un entorno controlado, he configurado Tuxeria usando Docker y Docker Compose. Esto permite levantar todo el proyecto con un solo comando y garantiza que la base de datos y el servidor web estén sincronizados y aislados del sistema host.

Servicios

  • tuxeria_db: contenedor MySQL 8 con la base de datos del proyecto.
  • tuxeria_web: contenedor PHP 8.2 + Apache que sirve los archivos de Tuxeria.

Volúmenes

El proyecto está completamente sincronizado con el contenedor web mediante volúmenes: 

./:/var/www/html

Esto permite que cualquier cambio en el proyecto local se refleje automáticamente en el contenedor.

Conexión PHP → MySQL

Dentro del contenedor web, la conexión se realiza usando el nombre del servicio MySQL definido en Docker Compose: 

$konexioa = mysqli_connect('tuxeria_db', 'root', 'rootpass', 'tuxeria');

Comandos para levantar y detener

Con Docker Compose, el proyecto se levanta y detiene fácilmente: 

docker-compose up -d --build
docker-compose down

Esto facilita la demostración del proyecto sin tocar la instalación local de PHP o MySQL.
