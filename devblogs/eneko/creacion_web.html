Creación de la página web de Tuxeria

14 de diciembre de 2025

En este artículo explico el proceso de creación de la página web de Tuxeria, una pizzería ficticia desarrollada como proyecto. El objetivo principal era crear una web funcional, clara y sencilla, utilizando tecnologías básicas del desarrollo web.

Estructura HTML

El HTML se ha utilizado para definir la estructura principal de la web. Cada página sigue una estructura clara y coherente para facilitar la navegación y el mantenimiento.

  • Formularios aplicados en todas las páginas para el tratamiento de datos posteriores con PHP.
  • Uso de etiquetas semánticas como <header>, <main> y <footer>.
  • Formularios para registro de clientes, pizzas y pedidos.
  • Páginas separadas para cada funcionalidad.
  • Uso de enlaces para la navegación desde el panel principal.

Estilos CSS

Para el diseño visual se ha utilizado CSS, buscando una estética sencilla, moderna y coherente en todas las páginas.

  • Diseño limpio con tonos azules.
  • Uso de Flexbox para organizar formularios y paneles.
  • Estilos reutilizables mediante clases comunes.
  • Diseño responsive para diferentes tamaños de pantalla.

Lógica PHP

PHP se ha utilizado para dotar a la web de funcionalidad dinámica y conexión con la base de datos MySQL.

  • Conexión a base de datos mediante mysqli.
  • Inserción de clientes, pizzas y pedidos.
  • Validación de datos introducidos en formularios.
  • Separación entre lógica y presentación.

Conclusión

El proyecto Tuxeria ha permitido aplicar conocimientos de HTML, CSS y PHP de forma práctica, creando una web funcional y organizada. Además, ha servido como introducción al desarrollo web completo (frontend + backend). Ha sido una buena experiencia de introducción al desarrollo WEB que recordaré con una sonrisa.
